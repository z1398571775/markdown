## 第一层：随机性 & 噪声（让世界不像机器）

### 🎲 随机与分布

| 算法                  | 一句话用途                       |
| --------------------- | -------------------------------- |
| Pseudo Random         | 生成可控随机结果                 |
| Seeded RNG            | 保证同种子结果一致               |
| Uniform Distribution  | 等概率随机                       |
| Gaussian Distribution | 制作“大多数普通、少数极端”的结果 |
| Poisson Distribution  | 控制稀疏事件发生                 |

| ---------------- | -------------------------------- |

| Worley Noise     | 细胞、裂纹、石头纹理             |
| Blue Noise       | 抗规律分布（采样）               |
| Pink Noise       | 自然节奏变化                     |
| Brown Noise      | 平滑运动路径                     |

# 📕 游戏算法终极字典（Final Form）

## 🧭 第一部分：按「游戏类型」反查算法

> **你在做什么游戏 → 你一定会用到哪些算法**

------

### 🎲 Roguelike / 地牢类

**核心问题**

- 地图随机但可玩
- 路线清晰
- 每局不同

**常用算法组合**

- 地图生成：
  - Cellular Automata（洞穴）
  - BSP（房间）
  - Drunkard Walk（随机路径）
- 变化控制：
  - Perlin Noise / FBM（装饰变化）
- 路径：
  - A* / BFS
- AI：
  - FSM / Behavior Tree
- 掉落：
  - Weighted Random / Loot Table

👉 **经典组合公式**

> Roguelike 地图 = BSP + CA + 噪声装饰

------

### ⚔️ ARPG / 动作游戏

**核心问题**

- 打击感
- 技能特效
- 怪物行为

**常用算法**

- 碰撞：
  - AABB / SAT / Raycast
- 运动：
  - Steering / Arrive / Ease
- AI：
  - Behavior Tree
  - Utility AI
- 特效：
  - 粒子系统
  - Curl Noise
- 数值：
  - DPS 曲线
  - Soft Cap

👉 **经典组合公式**

> 打击感 = 精准碰撞 + 插值运动 + 粒子反馈

------

### 🧠 RTS / 模拟 / 策略

**核心问题**

- 大量单位
- 群体移动
- 性能

**常用算法**

- 路径：
  - Flow Field
  - HPA*
- 群体：
  - Boids
  - RVO / ORCA
- AI：
  - Utility AI
  - GOAP
- 空间优化：
  - Quadtree
  - Spatial Hash

👉 **经典组合公式**

> RTS 移动 = Flow Field + 群体避让

------

### 🌍 开放世界 / 沙盒

**核心问题**

- 世界大
- 不重复
- 流式加载

**常用算法**

- 地形：
  - Perlin / Simplex
  - Voronoi
  - Erosion
- 生成：
  - WFC
  - Grammar-based
- 寻路：
  - NavMesh
  - HPA*
- 优化：
  - LOD
  - Chunk Streaming

------

### 🎯 塔防 / 放置 /数值驱动

**核心问题**

- 数值平衡
- 概率公平
- 路径明确

**常用算法**

- 路径：
  - BFS / A*
- 数值：
  - Logistic Curve
  - Soft Cap
- 概率：
  - Weighted Random
  - Alias Method
- 难度：
  - DDA
