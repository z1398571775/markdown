## 二、矢量

**点**:记录位置

**矢量**:标记着方向和大小,表示位移

### 2.1 矢量的加法和减法

想象两个点：

- 点A在(2, 1)
- 点B在(5, 5)

从A到B的方向和距离是多少？
`B - A = (5-2, 5-1) = (3, 4)`

这表示：**从A到B需要向右3个单位，向上4个单位**

```javascript
let posA = new cc.Vec2(2, 1);
let posB = new cc.Vec2(5, 5);
let directionAToB = posB.sub(posA); // cc.Vec2(3, 4)
```

### 实际应用：敌人追踪玩家



```javascript
update(dt) {
    let playerPos = this.player.position;  // 玩家位置
    let enemyPos = this.node.position;     // 敌人位置
    
    // 计算从敌人指向玩家的方向
    let direction = playerPos.sub(enemyPos); 
    
    // 归一化得到纯方向（长度为1）
    let normalizedDir = direction.normalize();
    
    // 敌人向玩家移动
    let move = normalizedDir.mul(this.speed * dt);
    this.node.position = this.node.position.add(move);
}
```
